## 👋 Building Toolsets for the Next-Gen Knowledge Base

Sa-lu-tations! I'm szktty.

I've recently created and maintain the following OSS libraries for Dart/Flutter:

- [Fonde UI](https://github.com/szktty/fonde-ui): A Desktop-first UI library optimized for instant feedback.
- [Plough](https://pub.dev/packages/plough): Interactive network graph visualization.
- [kiri-check](https://pub.dev/packages/kiri_check): Property-based testing for robust Dart code.
- [rinne_graph](https://pub.dev/packages/rinne_graph): An embedded graph database engine on top of SQLite.

I'm pouring my heart into [RinneGraph](https://github.com/szktty/rinne-graph-desktop) — a graph-based personal database tool that empowers ideas and creativity. I'v created the libraries for use within the app.
